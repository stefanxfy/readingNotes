### **<font style="color:rgba(0, 0, 0, 0.75);background-color:rgb(243, 243, 243);">一切的基石——实事求是的认知革命</font>**
+ “实事”就是客观存在着的一切事物，“是”就是客观事物的内部联系，即规律性，“求”就是我们去研究。  

+ 所谓实事求是，本质上就是要求我们的思想必须与客观现实精准地对应起来。  

+ 人类的大脑天生就喜欢走捷径，倾向于依赖直觉、经验、情绪和既有的观念来做判断。  

+ 我们看到一个商业模式成功了，就急于复制它的表象，却没有深入研究它成功的“实事”是什么，也就是它所处的市场环境、用户基础、技术条件以及它内部各个环节之间相互作用的“是”。我们听到一个耸人听闻的观点，就立刻被情绪调动，急于站队和批判，却没有去“求”这个观点背后的事实依据、数据来源和逻辑链条  

+ 这种基于主观愿望、僵化教条或者片面经验的思考方式，就是实事求是所要革命的对象。  

+ 真正的实事求是，第一步是放下预设的立场和答案。我们的大脑里装满了各种各样的“概念框架”，这些框架帮助我们快速理解世界，但同时也像一副有色眼镜，让我们看到的世界是经过过滤和扭曲的。  

+ 实事求是的第一课，是培养一种“空杯心态”，承认自己的无知，愿意像一个初生的婴儿那样，用不带偏见的眼睛去观察和感受这个世界的本来面貌  

+ 你要追问的第一个问题永远是：“事实究竟是什么？”而不是“我希望事实是什么？”或者“我认为事实是什么？”。  

+ 第二步，是深入地、全面地占有材料，也就是去做扎实的调查研究。这是“求”的核心环节。“没有调查，就没有发言权”，这句话的力量在于它斩断了一切空谈和臆想的根源。  

+ 你想了解一个市场，就要去和真实的用户聊天，看他们如何使用产品，听他们抱怨什么，观察他们的生活习惯。你不能只看报表上的数字，因为数字是“实事”的凝固结果，它不会告诉你用户为什么这样做。你想解决一个管理问题，就要深入到团队中去，和不同层级的员工交流，了解他们的工作流程、真实诉求和彼此之间的关系。  

+ 既要看有利的一面，也要看弊的一面；既要听赞成的声音，也要听反对的声音；既要了解事物的现状，也要追溯它的历史。片面的事实，往往比谎言更具欺骗性，因为它让你以为自己掌握了真相，实际上却把你引向了错误的深渊。  

+ 第三步，也是最关键的一步，就是对占有的材料进行“去粗取精、去伪存真、由此及彼、由表及里”的加工，从中找出事物内在的、必然的联系，也就是那个“是”。  

+ 运用逻辑分析、比较归纳、系统思考等工具  

+ 在个人成长上，它要求我们诚实地面对自己的优点和缺点，从自己真实的兴趣和能力出发规划人生，而不是盲目追逐社会上的热门标签。在企业经营上，它要求我们深刻洞察自己所处的产业环境、竞争格局和用户需求，围绕真实的价值创造来构建商业模式，而不是沉迷于编造动听的资本故事  

+ 在看待社会问题时，它要求我们超越情绪化的口号和简单的道德评判，去深入分析问题背后的结构性成因和各方利益纠葛。  

+ 掌握了实事求是，你就拥有了一个最可靠的罗盘，无论外界环境多么复杂多变，你都能找到通往正确方向的道路。这不仅是一种思想方法，更是一种正直、勇敢、谦逊的人格修炼。它是后续所有思维框架展开的基础，只有站在这块坚实的土地上，我们才能真正开始构建属于自己的、能够看透本质的思维大厦。  


### **<font style="color:rgba(0, 0, 0, 0.75);background-color:rgb(243, 243, 243);">解剖世界的利刃——阶级分析的思维方法</font>**
   


+ 它的本质，是根据人们在某个特定系统中所处的经济地位、所拥有的资源以及由此产生的根本利益，来对人群进行划分，并分析他们之间的相互关系和力量对比。  

+ 这个方法的核心在于，它认为驱动人类社会行为的最根本动力，不是抽象的道德、观念或者情感，而是实实在在的利益。  

+ 在一个组织或社会中，不同的人群由于其所处的位置不同，他们赖以生存和发展的方式不同，因此他们的核心利益诉求是不一样的。  

+ 在一家公司里，股东的核心利益是资本回报率和公司市值的增长；管理层的核心利益是在实现股东目标的同时，巩固自己的职位、获取更多的薪酬和权力；而基层员工的核心利益则是稳定的收入、良好的工作环境、个人技能的提升和未来的职业发展。这三者之间，既有合作的一面，比如公司发展好了，大家都能受益；但更存在着深刻的矛盾和博弈，比如，提高员工工资福利，就会减少股东的利润；加强内部管理控制，可能会限制员工的自由和创造力  

+ 阶级分析的思维，就是要求我们首先要能够清晰地辨认出这些不同的“利益群体”，理解他们各自的诉求和痛点。  

+ 在一个具体的“实事”面前，立刻在脑中画出一张“利益地图”。你要问自己这样几个问题：在这个局里，有哪些角色？谁和谁的利益是一致的或相近的，可以被划为一类？他们的核心利益诉求是什么？他们各自掌握着什么样的资源和权力？（比如，股东有资本，管理层有决策权和信息优势，员工有劳动力和执行能力）。他们之间的关系是怎样的？是相互依存、相互对立，还是可以相互转化？通过回答这些问题，原本一锅粥式的人际关系，就会立刻变得结构清晰、层次分明。你不再是看到一个个孤立的、情绪化的人，而是看到了他们背后所代表的那个群体的共同意志和行为逻辑。  

+ 举个例子，一个互联网平台想要修改规则，比如提高对内容创作者的分成门槛。如果我们仅仅从道德上评判平台“贪婪”，或者从情感上同情创作者，那就没有抓住问题的本质。运用阶级分析的思维，我们首先要识别出其中的几个关键群体：平台方、头部大V创作者、中腰部创作者、普通用户以及广告商。平台方的核心利益是商业利润和市场份额，修改规则是为了降低成本、提高效率。头部大V创作者拥有巨大的流量和粉丝基础，他们有较强的议价能力，甚至可以和平台博弈。中腰部创作者是这次规则修改最主要的受影响者，他们的生存空间被挤压，利益受损最严重。普通用户可能对规则修改无感，他们更关心内容的质量和多样性。广告商则关心平台的流量稳定性和投放效果。  

+ 平台和中腰部创作者之间存在直接的利益冲突，这是主要的矛盾。平台和头部大V之间是又拉拢又防备的复杂关系。平台需要讨好用户和广告商，因为他们是平台的生存基础。  

+ 于是，你就能预测接下来可能发生的事情：中腰部创作者会联合起来抗议，甚至出走；平台会一边安抚头部大V，给他们开出更好的条件，一边会用一些“算法推荐”之类的说辞来向用户和广告商解释规则的合理性  

+ 你要在公司内部推动一项改革，必然会触动一些人的利益，也必然会符合另一些人的利益。在你开始行动之前，你必须先用阶级分析的方法，把公司里的人分成三类：谁是改革的坚定支持者（改革会让他们获益），谁是改革的潜在阻力（改革会让他们受损或增加他们的不确定性），以及谁是摇摆不定的中间派。对于支持者，你要把他们团结起来，形成改革的核心力量。对于阻力，你要分析他们的担忧到底是什么，是害怕失去权力，还是害怕改变习惯，然后针对性地去化解，或者用阳谋去孤立其中最顽固的少数。对于中间派，你要做的就是争取他们，用改革成功后的美好前景和实际利益去打动他们，让他们倒向你这一边。  

+ 经典策略“团结朋友，分化敌人，争取中间”在现代管理中的应用  

+ 掌握阶级分析的思维，能让你拥有一双“X光眼”，看透温情脉脉的面纱背后，那些最真实、最坚硬的利益结构  

+ 一个人的态度和言论，并不由他的品德高下决定，而是由他所处的“位置”和他屁股下面的“椅子”决定的。  


  


### **<font style="color:rgba(0, 0, 0, 0.75);background-color:rgb(243, 243, 243);">驾驭复杂的罗盘——抓住主要矛盾的艺术</font>**
   


+ 当我们通过实事求是掌握了真实全面的材料，又通过阶级分析的方法认清了局中的各种角色和利益关系，我们往往会发现，问题不但没有变简单，反而变得更复杂了。各种各样的矛盾犬牙交错、盘根错节，千头万绪摆在面前，到底应该从何处着手  

+ 如果缺乏一个清晰的优先级判断标准，我们很容易陷入“眉毛胡子一把抓”的困境，最终被无穷无尽的事务拖垮，或者因为在次要问题上消耗了太多精力，而错过了解决核心问题的最佳时机。这时候，我们就需要第三件法宝，也是《毛选》思想精髓中的精髓，那就是：抓住主要矛盾。  

+ “矛盾”这个词，在《矛盾论》中有着深刻的哲学定义，它指的是事物内部普遍存在的对立统一关系。  

+ 任何事物、任何过程中，都充满了各种各样的斗争和差异，这些就是矛盾。比如，一个企业内部，有发展和稳定之间的矛盾，有创新和守成之间的矛盾，有短期利益和长期利益之间的矛盾，有不同部门之间的本位主义矛盾，等等。  

+ 承认矛盾的普遍性，是我们分析问题的第一步。  

+ 如何判断什么是主要矛盾呢？标准只有一个：看哪个矛盾的存在和发展，规定或影响了其他矛盾的存在和发展。主要矛盾是那个“牛鼻子”，牵一发而动全身。  

+ 在抗日战争时期，中华民族面临着无数的矛盾：有中华民族和日本帝国主义的民族矛盾，有国内的阶级矛盾（地主与农民、资产阶级与无产阶级），还有国民党内部不同派系的矛盾，等等。在当时，哪一个才是主要矛盾？毫无疑问，是民族矛盾。因为日本的侵略威胁到了整个民族的生存，这个矛盾的尖锐性压倒了一切。如果这个问题不解决，国内的其他矛盾都无从谈起。所以，当时提出“停止内战，一致对外”，建立抗日民族统一战线，就是抓住了主要矛盾的英明决策。把解决其他次要矛盾（比如土地革命）暂时服从于解决主要矛盾这个大局，这就是高超的战略思维。  

+ 。而高手则会把百分之八十的精力，像激光一样聚焦在那个百分之二十的核心问题上  

+ 一个人想要提升自己，可能发现自己有很多缺点：拖延、表达能力不强、专业知识不足、人脉不广……如果你试图同时改进所有这些，你很可能会因为看不到明显的进步而感到挫败，最终放弃。你应该做的，是分析在当前你的职业或人生阶段，哪个短板最限制你的发展，哪个就是你的主要矛盾。  

+ 识别主要矛盾并非易事，它需要深刻的洞察力和全局观。它要求我们不能只看树木，不见森林；不能只看眼前，不看长远。  

+ 一个团队表面上看是执行力不强，但深挖下去，可能会发现主要矛盾是团队成员对目标缺乏共识，大家不知道为何而战，所以才没有动力。这时候，你去抓考勤、抓KPI，都是在次要矛盾上用功，真正需要做的，是开一次触及灵魂的战略共识会，把“为何而战”这个问题彻底解决掉。  

+ 抓住主要矛盾的过程，本身就是一个不断深化认识、逼近事物本质的过程。它建立在实事求是和阶级分析的基础之上，是前两者逻辑发展的必然结果。  


  


### **<font style="color:rgba(0, 0, 0, 0.75);background-color:rgb(243, 243, 243);">世界运动的引擎——矛盾的对立统一法则</font>**
   


+ 世界运动变化的根本规律，也就是所有矛盾运动的内在引擎——对立统一法则  

+ 理解了它，我们就不再是静止地、孤立地、片面地看待问题，而是能够动态地、全面地、联系地洞察万事万物的本质  

+ 所谓对立统一，简而言之，就是任何矛盾的双方，既是相互排斥、相互斗争的（对立），又是相互依存、相互联结，并在一定条件下可以相互转化的（统一）。这个法则揭示了事物发展的根本动力，不是来自外部的推动，而是源于其内部矛盾的运动。  

+ 理解“对立”的绝对性  

+ 斗争是无时无刻不存在的。这不仅指战争、冲突这种激烈的形式，更泛指一切形式的差异、排斥和相互作用。  

+ 在人体内部，细胞的生长和凋亡始终在斗争；在市场上，买方希望价格越低越好，卖方希望价格越高越好，这之间存在永恒的对令；在一个团队里，追求创新的力量和维持稳定的力量总是在相互博弈。正是这种斗争，才构成了事物的生机与活力，推动着事物不断向前发展。一个没有了任何矛盾和斗争的组织，也就成了一潭死水，离消亡也就不远了  

+ 我们要敢于承认矛盾、正视矛盾，甚至在某种程度上要欢迎矛盾，因为它是变革和进步的信号  

+ 统一”体现在两个方面：其一是相互依存。矛盾的双方，谁也离不开谁，一方的存在以另一方的存在为前提。没有黑暗，就无所谓光明；没有失败，也就无所谓成功；没有风险，也就没有收益  

+ 在企业里，生产部门和销售部门看似矛盾重重，生产部门抱怨销售部门的预测不准，销售部门抱怨生产部门的交期太慢，但他们彼此又是唇亡齿寒的关系，没有销售，生产就没有意义；没有生产，销售就是空中楼阁。  

+ 解决矛盾的目的不是要消灭某一方，而是要让双方在一个更高水平上达成新的平衡。  

+ “统一”的第二个方面，也是最激动人心的，是相互转化。在一定的条件下，矛盾的双方可以向其对立面转化。劣势可以转化为优势，危机可以转化为机遇，失败可以孕育成功，被领导者也可以变成领导者。  

+ 所谓“福兮祸所伏，祸兮福所倚”，讲的就是这个道理。对立统一的思维，就是教我们始终要用一分为二的眼光看问题，在顺境中看到潜在的危机，在逆境中找到翻盘的希望。  

+ 掌握了对立统一的法则，我们看问题的视角就会变得立体和富有弹性  

+ 当我们看到一个问题时，我们会习惯性地去分析它对立面的积极因素是什么；当我们取得一个成就时，我们会警惕地审视其中潜藏的风险。比如，一项技术革新，它既可能带来生产效率的巨大提升（好的一面），也可能导致大量工人失业，引发社会问题（坏的一面）。一个全面的决策，必须同时考虑到这两个方面，并设计出相应的配套措施，去放大其利，规避其害  

+ 对立统一法则是我们理解世界复杂性和动态性的终极钥匙。它让我们超越了非黑即白的线性思维，进入了一个充满可能性和转化空间的辩证世界。它要求我们有全局观，不能只看一面，而要看到矛盾的双方；它要求我们有发展观，不能静止地看问题，而要看到双方力量的消长和转化的可能；它要求我们有条件观，要积极地去创造那些能促使矛盾向有利方向转化的条件。  

+ 任何危机的背后都潜藏着转机，任何强大的对手身上都存在着可以被利用的弱点。  


  


### **<font style="color:rgba(0, 0, 0, 0.75);background-color:rgb(243, 243, 243);">以弱胜强的宝典——持久战的战略智慧</font>**
   


+ 在矛盾的双方中，我们恰恰是处于弱势的那一方。无论是在商业竞争中，一家小公司面对行业巨头；还是在个人成长中，一个初出茅庐的年轻人想要追赶资深的前辈；抑或是在一场艰苦的谈判中，我方掌握的筹码远少于对方  

+ ？《论持久战》这部伟大的著作，虽然是为了一场具体的战争而写，但它所揭示的以弱胜强的底层逻辑，为所有处于劣势地位的奋斗者提供了一部颠扑不破的战略宝典。其核心思想，就是反对速胜论的盲动和亡国论的悲观，主张在客观认识敌我优劣的基础上，通过发挥主观能动性，打一场有计划、有步骤、以时间换空间的持久战。  

+ 持久战的第一个思维精髓，是客观、清醒地评估“基本条件”。在任何对抗开始之前，必须进行冷静的“SWOT分析”，也就是分析敌我双方的优势（Strengths）、劣势（Weaknesses）、机会（Opportunities）和威胁（Threats）  

+ 在《论持久战》中，对中日双方的矛盾特点做了极其深刻的分析：日本是强国，武器装备先进，这是它的优势；但它是非正义的侵略战争，国内资源有限，国际上失道寡助，这是它的劣势。中国是弱国，技术和军力落后，这是我们的劣势；但我们是正义的反侵略战争，地大物博，人口众多，能获得国际同情，这是我们的优势。基于这种清醒的认识，得出的结论必然是：速胜是不可能的，因为敌强我弱；亡国也是不可能的，因为敌小我大、敌退步我进步、敌寡助我多助。这个结论，就为整个战略奠定了最坚实的认知基础  

+ 对方的强大背后，隐藏着哪些固有的、难以克服的弱点？你自身的弱小之中，又蕴含着哪些可以被利用的、决定未来的优势？

+ 第五，关于策略（持久战/群众路线）：我的长期目标是什么？当前我处于哪个阶段？我应该采取什么样的策略，是该防御积累，还是该主动进攻？我需要依靠谁的力量来实现目标？我如何去倾听他们的声音，获得他们的支持？  

+ 第六，关于行动（实践论）：为了验证我的想法，我能做的最小成本的实践是什么？我应该如何设计我的行动方案，以便能最快地获得反馈？我从上一次的行动（或失败）中学到了什么？  

+ 你能够比别人更快地看透事物的本质，更准确地预测事物的走向，更有效地找到解决问题的杠杆点。  

+ 实事求是要求你谦逊和诚实，勇于面对自己不愿承认的现实。阶级分析让你对人性有更深刻的洞察，从而变得更加通透和包容。抓住主要矛盾锻炼你的决断力和战略定力。对立统一赋予你在变局中保持乐观和弹性的智慧。持久战培养你无与伦比的耐心和韧性。群众路线让你学会倾听与共情。调查研究塑造你严谨务实的工作作风。而实践论，则给了你直面失败、从头再来的勇气。
